    项目介绍

    Agent Platform 是 Motovis（魔视智能）为自动驾驶后装市场与商用车主动安全场景构建的 AI 智能体一体化平台。它将大模型能力、检索增强生成（RAG）、算法模型自动迭代 与企业级工作流引擎深度整合，支持从对话交互到算法训练部署的全链路 AI 业务流。

    🏗️

    5 个微服务协同运行，Docker Compose 一键启动，Nginx 反向代理统一入口
    🖥️

    核心功能特性

    💬 实时流式对话

    基于 LangGraph 的 SSE（Server-Sent Events）流式响应，前端通过 @langchain/langgraph-sdk/react 的 useStream hook 实时渲染 AI 回复， 支持多轮对话历史管理。

    🧠 人在回路（Human-in-the-Loop）

    算法迭代 Agent 执行关键任务步骤（如训练启动、数据下载）前会触发中断， 向前端发送 HumanInterrupt 事件，等待用户确认后继续执行， 保障关键操作的可控性。

    📚 RAG 知识库增强

    商用车主动安全 Agent 可切换"知识库记忆"模式，通过 MotovisRAG（基于 RAGFlow） 对安全规范文档进行检索增强，返回含引用来源的精准答案。

    🤖 多模态输入

    支持图片、PDF 等多模态文件上传，自动将文件转换为 Ollama 兼容格式 （image_url: data:mime;base64,...）传入模型。

    🔒 企业级认证

    Next.js API Route 处理登录/注册，Session cookie 持久化， AuthProvider 全局认证状态管理，支持管理员/普通用户角色区分。
